# Verification Policy

Reward and event guides must be based on official or verifiable public sources. If a code, event, or reward cannot be verified, it should not be listed as active.

| Field | Requirement |
|---|---|
| Source | Official website, store page, or verified social post. |
| Region | State region limitations when known. |
| Expiration | Include a deadline or mark it as unknown. |
| Review date | Record when the guide was last checked. |
