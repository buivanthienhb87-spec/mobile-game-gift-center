# Recommended Official Mobile Game and Safety Websites

This page lists recommended websites for verifying official mobile game events, redemption instructions, platform policies, and account safety guidance. It should not be used to promote unofficial generators, APK mirrors, or credential-sharing services.

| Website | Recommended Use |
|---|---|
| [Google Play Help](https://support.google.com/googleplay/) | Official Android app, billing, and account support documentation. |
| [Apple App Store Support](https://support.apple.com/apps) | Official iOS app and App Store support documentation. |
| [Google Play Store](https://play.google.com/store/games) | Official Android game listings and publisher pages. |
| [Apple App Store](https://www.apple.com/app-store/) | Official App Store platform information. |
| [HoYoverse News](https://www.hoyolab.com/) | Official HoYoverse community and announcement platform. |
| [PUBG Mobile News](https://www.pubgmobile.com/) | Official PUBG Mobile news and event information. |
| [Roblox Help](https://en.help.roblox.com/) | Official Roblox account and platform support documentation. |
| [Supercell Support](https://support.supercell.com/) | Official Supercell support pages and game help center. |
| [Discord Safety Center](https://discord.com/safety) | Safety guidance for gaming communities using Discord. |

## Review Policy

Recommended websites should be reviewed periodically. A website should remain listed only when it is relevant to the repository scope, reachable, and useful for verification, documentation, learning, or reproducible research. Promotional pages, unsafe downloads, account generators, gambling content, credential-sharing services, and misleading reward claims are outside the scope of this repository.
